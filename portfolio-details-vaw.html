Violence Against Women: Linear Regression for Causal Inference

  • Category: Class Project
  • Class: Data and Society
  • Date: Fall 2021
  • GitHub URL: https://github.com/AlisonYao/Violence-Against-Women

Description

We conducted data analysis and data visualization on 3 indicators of violence against women (VaW) and utilized linear regression to find poverty level as a potential confounder to the spurious correlation between the prevalence of VaW and alcohol consumption.
